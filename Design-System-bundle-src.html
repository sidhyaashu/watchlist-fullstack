InvestKaro Design System

A glassmorphism-led system for a personal investment advisor. Soft blue paper in light, charcoal & black in dark — both anchored on translucent surfaces, generous spacing, and editorial-grade typography. Built for analytical density without losing warmth.

01

Principles

Five rules that explain why every screen looks the way it does.

P.01

Calm density

Show many numbers without alarm. Use whitespace, neutral ink, and tight monospace alignment so dense data feels like a quiet ledger — not a trading desk.

P.02

Glass over chrome

Surfaces are translucent layers, not painted boxes. The background gradient breathes through every panel; chrome is implied by hairlines and inset highlights.

P.03

Editorial first

Inter for UI, JetBrains Mono for numbers. Headlines tighten letter-spacing and use ink-1; everything else steps down through ink-2/3/4.

P.04

Two faces, one bone

Light is soft-blue paper; dark is charcoal & black. Tokens swap, structure does not. The same component reads at home in either mode without a redesign.

P.05

Color earns its keep

Accent blue is reserved for primary actions and active state. Green/red appear only on financial change. Anything else is ink — full stop.

P.06

Motion is feedback

Transitions are ≤180ms, eased. Animations explain state — never decorate. Background blobs drift slowly enough to feel ambient, not active.

09

Using the system

Drop the token block at the top of your stylesheet, mount the toggle, and every component below inherits the theme.

Token block
CSS variables
/* light is the default */
:root{
  --ink:#0B2545;       /* headings, key data */
  --accent:#2B6BFF;    /* primary actions */
  --good:#10A37F;      /* up · positive */
  --danger-deep:#C0365B;/* down · text */
  --mono:'JetBrains Mono', ui-monospace, monospace;
  --r-md:12px;
}

/* dark just remaps */
[data-theme="dark"]{
  --ink:#F2F2F3;
  --accent:#E5E5E7;
  --good:#4ADE80;
  --danger-deep:#FCA5A5;
}
Theme toggle (vanilla JS)
~10 lines, persists to localStorage
// init: respect saved preference, fall back to OS
const saved = localStorage.getItem('ik-theme');
const sysDark = matchMedia('(prefers-color-scheme: dark)').matches;
document.documentElement.setAttribute('data-theme', saved || (sysDark ? 'dark' : 'light'));

// toggle
themeBtn.onclick = () => {
  const cur = document.documentElement.getAttribute('data-theme') || 'light';
  const next = cur === 'dark' ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', next);
  localStorage.setItem('ik-theme', next);
};
